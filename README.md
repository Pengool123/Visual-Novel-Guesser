# Visual Novel Guesser

[Play here](https://pengool123.github.io/Visual-Novel-Guesser/)<br>

![Blank look](githubImgs/initLook.png)
<br>
Guess the visual novel based on the hints given and tag overlaps from guessed visual novels.<br>

7 total tries per game<br>
- It will first give 5 random tags to start off<br>
<br>
Hint 1 - Available languages<br>
Hint 2 - The popularity range<br>
Hint 3 - The ranking range<br>
Hint 4 - Best quote if available, rough length, and if there is an Eroge version<br>
Hint 5 - A blurred image of the title<br>
One more guess is given after the last hint<br>

![Winning screen](githubImgs/win.png)
<br>
Note: the VNDB API's length isn't fully accurate compared to what's shown on the website<br>
Example: Katawa Shoujo shows as 'Long' on the website, but the API deems is 'Medium'<br>
<br>

## Features

![Settings screen](githubImgs/settings.png)

- Include Eroge visual novels or not
- Include Eroge tags
- Change the spoiler level of what tags will and wont be shown
- Unblur the image even if it's sexual or explicit depending on settings
- Unblur the image even if it's violent or brutal depending on settings
